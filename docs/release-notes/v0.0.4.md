
## Known limitation: unsigned installers

**v0.0.4 installers are not code-signed.** macOS will trigger Gatekeeper warnings ("can't be opened because Apple cannot check it for malicious software"); Windows will trigger SmartScreen warnings. Installers can be modified in transit without detection. Code signing and notarization are tracked in [M5 — Code Signing & Distribution Security](https://github.com/manomatika/ahimsa/milestone/10) and will land in a later release. **Do not distribute these installers to external users.**
